# 先遣故事一
[探照灯]队先遣故事：
“呜诶！？我的灯哪去了？”一个少女神色慌张地翻找着自己的背包。
“那个……”一位戴着眼镜的高个子男一脸无语看着“灯……不就在你旁边吗？”
“啊！找到了——”
“唉，这个人真的能当我们队长吗？”高个子男嘀咕道。
“没办法，老队长的继承人，将就一下吧……”一个十分强壮的男人把手搭在高个子男肩膀上。
“但是，你们不觉得她有点像老队长吗？”一个女人轻柔地说到。
“哪里像了……”

[探照灯]队先遣故事二：
“本叔，饭什么好，我好饿啊～”少女捂着干瘪的肚子。
“快了快了，别着急！”卡尔.本勺起锅里的一勺汤试了试味道。
“不知道今天本叔能做出什么好吃的？炖肉？烤肉？”一向温柔的丁呦呦也开始幻想起来。

# 先遣故事三
[探照灯]队先遣故事三：
呼——呼！！——
“山叔！呦呦姐！你们在哪？”铃冒着狂风暴雪举起探照灯，但灯光似乎穿透不了前面的暴雪，“该死，喂！戴眼镜的，接下来该怎么走？”
铃往前塌了一步，突然感受到连接着迭戈的绳子瞬间绷直，“戴眼镜的？”
她回头一看，发现迭戈已经倒再白雪之中。
“喂！戴眼镜的！别再这睡啊！”铃顺着绳子来的迭戈旁边，然后不停的扇巴掌，“喂！臭戴眼镜的！别睡着啊！喂！迭戈！”

# 先遣故事四
[探照灯]探险队先遣故事四：
“不！爸爸！妈妈！不要离开我！”丁呦呦猛然从床上惊醒，大口地喘着气，眼角也有不少泪痕。
“又做噩梦了？”还在捣鼓机械瓦尔特.山过来给丁呦呦搭上一件衣服。
“山叔，我爸爸妈妈为什么要离开我？是我有什么做的不好吗？”
瓦尔特.山看着稚嫩的丁呦呦，一时半会不知道说什么，这个年纪的她如果知道真相，估计会疯掉，“等你长大了，你自然会知道……”
“等我长大了，我自然会知道……山叔，你这句话还真让我记住很久啊。”一个温柔的声音在瓦尔特.山耳边响起。
他看着已经长得亭亭玉立的丁呦呦，此时的依靠在墙边眺望云海，他突然觉得一阵恍惚，“这下，你不是知道了吗？”
丁呦呦手旁边，一本泛黄的笔记被一阵风吹翻页起来。
“快走吧，今天还要见另一个丫头。”
“嗯？另一个丫头？”
“那是我另一位老友女儿……”

# 先遣故事五
[探照灯]探险队先遣故事五：
“铃啊，外面的世界很危险，探险家什么的就别去了。”铃的妈妈语重心长地对铃说。
“况且，爸爸我已经替你打点好关系了，等你到了工作的年龄，就可以在中央城邦干舒舒服服安全的文工了。”
铃堵着小嘴背过身去，气呼呼地说：“我不要！我就要去当探险家！况且爸爸妈妈不就是因为去探险认识的吗？还是爷爷撮合的你们！”
夫妇俩听到着瞬间无语，也背过身悄悄商量起来：“爸怎么把这事也给铃说啊”
“铃这性格还真和咱爸像，都是属驴的……”
“有了，得让她知难而退！让她经历一次小冒险，这样就会知道当探险家的辛苦了！”妈妈突然灵光一现。
“对了，可以让山哥帮忙……”
“这种事麻烦山哥干嘛？”
“诶，你听我说……”

# 先遣故事六
[探照灯]探险队先遣故事六：
“喂！臭戴眼镜的！别睡啊！”铃狂扇迭戈的耳光，“山叔！呦呦姐！你们在哪？！”
其实她很清楚，爷爷笔记记载这样不停的大喊很浪费体力，特别在这种场合……但，这是唯一的希望了……
“铃！迭戈！你们在哪？”此时瓦尔特.山的声音传入了铃的耳朵。
“是山叔！？”铃举起提灯不停的晃悠，“山叔！我们在这！山叔！！”
瓦尔特.山找到了铃她们，并带她们来到一处小树林，这里的狂风小了很多，所以瓦尔特.山和丁呦呦在这搭了一个临时营地。
“不好！迭戈身体失温了！”丁呦呦检查着迭戈的身体。
“失温！？”铃二话不说脱掉自己和迭戈的半湿不冷的衣服然后一把抱住迭戈，丁呦呦把瓦尔特.山和她自己的大衣盖在两人身上，还往里面塞了一下干的杂草。
随后丁呦呦和瓦尔特.山出去找柴火。
不知道过了多久，迭戈迷迷糊糊的醒了过来。
“醒了？”瓦尔特.山拨弄着自己跟前烧的正旺的火堆。
“我……这是……”迭戈低头一看，发现铃赤裸着抱着赤裸的自己，嘴边还耷拉着口水，看到着迭戈脸瞬间红到后耳根。
“来，把这个莱姆水喝了，暖暖身子”
“山叔，我睡了多久？”迭戈看了看睡在火堆旁的丁呦呦。
“不知道……表在大风中丢失了，庆幸的是，风小了很多，不幸的是，物资丢了不少。”

# 先遣故事八
[探照灯]探险队先遣故事八：
几人拿着灯往自己脚下的深渊照去，“山叔，丢一根荧光棒下去。”
“好。”瓦尔特.山掰开一根荧光棒忍下深渊，但很快，荧光棒那微弱的光被黑暗吞没，“我的老天，这么深？”
瓦尔特.山话音刚落，地面突然开始剧烈震动，几人重心不稳趴在地上。
而那片深渊也随着震动发出很奇怪的类似咯吱咯吱的声音，似乎有什么东西正在爬出深渊……
“靠，那是什么怪物！？”丁呦呦举起手中的探照灯照向从深渊爬出来的怪物。
“喔！喔！喔！那怪物要朝我们压过来了！快跑！！”
几人赶紧爬起往远离怪物的方向跑去。
